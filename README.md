# docker-geoserver

This container is deploying [GeoServer] over [Apache Tomcat], through its
[Web Archive] package.

Currently, versions are:
* GeoServer: `2.14.2`
* Tomcat: `8.5.37`

[geoserver]: http://geoserver.org/
[apache tomcat]: http://tomcat.apache.org/
[web archive]: http://geoserver.org/release/stable/
